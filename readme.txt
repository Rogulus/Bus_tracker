Bus tracker - aplikace pro sledování pohybu autobusů na mapě.


Autoři: Marek Šťastný (xstast33), Michael Kinc (xkincm02)

Sestavení projektu provedete v hlavním adresáři Bus_tracker zadáním příkazu:
    ant compile

Spuštění projektu provedete v hlavním adresáři Bus_tracker zadáním příkazu:
    ant run
